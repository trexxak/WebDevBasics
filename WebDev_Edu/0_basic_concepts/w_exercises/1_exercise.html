HTML Cheat Sheet

HTML Elemente und ihre Funktionen

  • <h1> bis <h6> - Überschrift.
    Variiert in Größe, wobei h1 die oberste Überschriftenebene darstellt.

  • - Paragraph.
    Beginnt immer in neuer Zeile. <ul> - Ungeordnete Liste.
    Beinhaltet mehrere List-Item Elemente. Listenpunkte auf selber Ebene werden durch gleichbleibendes Zeichen angeführt.
  • <ol> - Geordnete Liste.
    Beinhaltet mehrere List-Item Elemente. Listenpunkte auf selber Ebene werden durch inkrementierende Zeichen (numerisch, alphabetisch, etc.) angeführt.

  • <li> - Listenpunkt.
    Wird sowohl in geordneten- als auch in ungeordneten Listen verwendet. Kann selbst eine Liste beinhalten, welche dann eine Ebene tiefer dargestellt wird.


    • <img> - Bild.
    Der Pfad zum Bild muss als Attribut "src" angegeben werden.
  • <a> - Verknüpfung.
    Die Zieladresse muss als Attribut "href" angegeben werden.
